# LLVM Qualification Working Group meetings

This directory contains materials related to the LLVM Qualification Working Group's monthly meetings.

- `materials/` contains the meeting materials in Markdown format.
- `img/` contains images referenced by the meeting materials.
- `minutes/` is intended for meeting minutes.

Meeting material files follow the naming convention: `YYYYMM_llvm_qual_wg.md`
