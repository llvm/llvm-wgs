# LLVM Qualification Working Group meetings

This directory contains materials related to the LLVM Qualification Working Group's meetings.

## Directory structure

* `materials/` contains the meeting slide content in Markdown format.
* `minutes/` contains published meeting agendas and minutes in Markdown format.
* `img/` contains images referenced by the meeting materials.

## File naming

Regular monthly meeting materials follow these naming conventions:

* Meeting materials: `YYYYMM_llvm_qual_wg.md`
* Agenda: `YYYYMM_llvm_qual_wg_agenda.md`
* Minutes: `YYYYMM_llvm_qual_wg_minutes.md`

Files for special meetings or sessions may include the full date and a short topic identifier, for example:

```text
YYYYMMDD_<topic>.md
```

## Sources

* [LLVM Qualification WG meeting materials](https://drive.google.com/drive/folders/1nu3JAanE0gqQDll0S9ofVy4FOFezc6Mm)

* [LLVM Qualification WG sync-ups meeting minutes](https://discourse.llvm.org/t/llvm-qualification-wg-sync-ups-meeting-minutes/87148)

The Markdown files in this repository preserve the published wording while adapting the original formatting for long-term storage in the repository.
